Listado de categorías, búsqueda y estado.

Se ha seleccionado el menú desplegable "Tienda" accediendo al área de categorías. Uml del programa


Ver UML

Blaspoint implementa las siguientes características:

  1. Posee una programación orientada a objetos
  2. Posee una clase padre (herencia) y a la vez es abstracta
  3. Contiene 2 interfaces CRUD (listas de acciones de un determinado objeto)
  4. Blaspoint tiene acceso a una base de datos MYSQL en PHP My Admin
  5. Interfaz Gráfica de Usuario (ventanas)

El programa está escrito utilizando los patrones MVC y DAO.
